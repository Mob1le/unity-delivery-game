| Project Concept |  |  |  |
| :---- | :---- | :---- | :---- |
| **1 Player Control** |  | You control a  *Delivery man*  | in this   *Top Down 3D* game  |
|  |  | where *WASD or arrow keys*  | makes the player *Walking on the ground*  |

| 2 Basic  Gameplay |  | During the game,  *Rocks and wild birds* appear  | from  *Sky in the ground randomly*  |
| :---- | :---- | :---- | :---- |
|  |  | and the goal of the game is to  *To reach the villages for deliver supply and buy some usefull staff*  |  |

| 3 Sound  & Effects |  | There will be sound effects  *Footsteps*  | and particle effects *Rocks hit the ground, wild birds catch the player*  |
| :---- | :---- | :---- | :---- |
|  |  | \[*optional*\] There will also be *Simple background music, sun or night insects*  |  |

| 4 Gameplay Mechanics |  | As the game progresses,  *Delivery man walks on the road or offroad*  | making it *Try to avoid dangerous falling rocks and wild birds*  |
| :---- | :---- | :---- | :---- |
|  |  | \[*optional*\] There will also be *Delivery man have a health and provisions bar. If he is caught, a health bar is spent. Also, provisions are periodically spent and need to be replenished in the villages.*  |  |

| 5 User Interface |  | The  *provisions*  | will  *decrease*  | whenever *timer*  |
| :---- | :---- | :---- | :---- | :---- |
|  |  | At the start of the game, the title *Main menu* will appear  |  | and the game will end when *Reach the town*  |

| 6 Other Features |  |  Feature 1. Player can upgrade his skills on villages ( improve max weight, speed, discover other skills like jump or strafe which help for avoid enemies  |  |
